# Öğrenci Yönetim Sistemi

Python ve SQLite kullanılarak geliştirilmiş terminal tabanlı öğrenci yönetim sistemi.

## Özellikler
- Öğrenci ekleme, listeleme, arama, güncelleme ve silme
- Bölüm ve sınıfa göre istatistik raporu
- Hata kontrolü ve silme onay mekanizması

## Kullanılan Teknolojiler
- Python 3
- SQLite (sqlite3 modülü — harici kurulum gerekmez)

## Çalıştırma
```bash
python main.py
```

## Menü
```
1 - Öğrenci Ekle
2 - Öğrencileri Listele
3 - Öğrenci Ara
4 - Öğrenci Güncelle
5 - Öğrenci Sil
6 - İstatistikler
0 - Çıkış
```
